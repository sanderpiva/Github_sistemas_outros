
Inserindo projetos para o github

1- No cmd
2- cd [ nome pasta que tem o projeto ]
3- git init
4- git status
5- git add .
6- git status
7- git commit -m 'projeto'
8- git remote add origin [ caminho do respositorio do githun com '.git' no final ]
9- git push origin master

Para atualizar um projeto ja no repositorio github

1- No cmd
2- cd [ nome pasta que tem o projeto ]
3- git init
4- git status
5- git add .
6- git status
7- git commit -m 'projeto'
8- git pull origin master
9- git push origin master
